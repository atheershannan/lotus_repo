## 🎯 דוגמה משלחת: Task Manager App

### Stage 01 - שאלות לדוגמה
**שאלה 1:** מה שם הפרויקט?  
**תשובה:** "Task Manager App"

**שאלה 2:** איזו בעיה יפתור?  
**תשובה:** "אנשים מתקשים לעקוב אחרי המשימות שלהם ולזכור מה עליהם לעשות"

**שאלה 3:** מי קהל היעד?  
**תשובה:** "משתמשים פרטיים שמחפשים כלי פשוט לניהול משימות"

**שאלה 4:** מה תכונות החובה (MVP)?  
**תשובה:** "יצירת משימה, מחיקת משימה, עדכון סטטוס, הצגת כל המשימות"

**שאלה 5:** מה תכונות לעתיד?  
**תשובה:** "קטגוריות, תאריכי תפוגה, תזכורות"

**שאלה 6:** מה המגבלות?  
**תשובה:** "תקציב 0$, זמן 2 חודשים, צוות 1 מפתח"

**שאלה 7:** אימות משתמשים?  
**תשובה:** "כן, JWT עם email/password"

**שאלה 8:** איזה מידע יישמר?  
**תשובה:** "משתמשים, משימות (כותרת, תיאור, סטטוס, תאריך)"

**שאלה 9:** תשלומים?  
**תשובה:** "לא ב-MVP"

**שאלה 10:** התקנים נתמכים?  
**תשובה:** "Web responsive (Desktop + Mobile)"

## 📝 דוגמת פקודה להתחלה

```
צור פרויקט חדש בשם "Task Manager" לפי FULLSTACK_TEMPLATES/Stage_01_Requirements_and_Planning/REQUIREMENTS_TEMPLATE.prompt

אני רוצה ליצור Task Manager ש:
- יאפשר למשתמשים ליצור ולמחוק משימות
- יהיה עם authentication
- עבור למסד נתונים פשוט (SQLite או PostgreSQL)
- תקציב 0$ (open source stack בלבד)
- זמן פיתוח 2 חודשים

השתמש ב-10 השאלות מה-REQUIREMENTS_TEMPLATE.prompt כדי ליצור requirements.md
```
